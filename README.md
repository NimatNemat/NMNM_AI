## 소개
니맛내맛 프로젝트는 사용자의 식당 평가를 기반으로 식당을 추천해주는 시스템입니다. 사용자는 자신이 선호하는 식당 그룹에 따른 추천, 사용자의 식당 평가에 따른 개인 별 추천, 마지막으로 함께 먹을 사람을 추가하여 함께 먹는 인원 모두의 취향에 맞는 식당을 추천해줄 수 있는 함께 먹기 추천 방식을 제공합니다.
